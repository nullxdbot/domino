Tentang Aplikasi

Domino Score NullXD

Version 1.0.0

Aplikasi pencatat skor domino modern dengan fitur lengkap dan antarmuka yang elegan untuk pengalaman bermain yang lebih menyenangkan.

Fitur Utama

  • Score tracking untuk 2 pemain
  • Calculator terintegrasi
  • History tracking per ronde
  • Win counter & last winner badge
  • 3 pilihan tema warna
  • Sound effects & compact mode
  • Data tersimpan otomatis (LocalStorage)
  • Anti-sleep mode (Wake Lock)
  • PWA ready - Install ke home screen
  • Animated particles background

Cara Penggunaan

1
Menambah Skor

Klik area skor atau tombol untuk membuka calculator

2
Mengurangi Skor

Klik tombol untuk mengurangi 1 poin

3
Reset Pemain

Klik tombol di pojok card untuk reset skor pemain

4
Ganti Tema

Buka Settings dan pilih tema favorit Anda

5
Install App

Klik tombol "Add to Home Screen" di browser untuk akses cepat
